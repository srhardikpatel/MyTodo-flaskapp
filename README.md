# DevOps Project: CI/CD Pipeline for a 2-Tier Flask Todo Application (Flask + MySQL)

## 1. Project Overview

This project uses a CI/CD pipeline implemented with GitHub Actions to automate the build and deployment process.

This document includes step-by-step process of deploying a 2-tier web application (Flask + MySQL). 

A CI/CD pipeline is configured using GitHub Actions to automatically build a Docker image, push it to Docker Hub, and update the deployment file with the latest image tag whenever new code is pushed to the GitHub repository.
